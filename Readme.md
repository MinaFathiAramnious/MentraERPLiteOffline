## 📖 عن المشروع
**Mentra ERP Lite** هو تطبيق ويب تقدمي (PWA) صُمم خصيصاً لأصحاب الأنشطة التجارية الصغيرة والمتوسطة الذين يبحثون عن نظام سريع، آمن، ومجاني لإدارة مبيعاتهم ومخازنهم دون الحاجة لدفع اشتراكات شهرية أو الاعتماد على اتصال دائم بالإنترنت. 

يتم حفظ جميع البيانات محلياً على جهاز المستخدم لضمان أقصى درجات الخصوصية والسرعة.

---

## ✨ المميزات الرئيسية

* 🛒 **إدارة المبيعات والمشتريات:** إنشاء فواتير سريعة ومرنة.
* 📦 **إدارة المخزون الذكية:** تتبع فوري للكميات مع تنبيهات تلقائية للنواقص (الحد الأدنى للمخزون).
* 📊 **لوحة تحكم تفاعلية (Dashboard):** إحصائيات لحظية للمبيعات، وقيمة المخزون، وفلترة ذكية بناءً على التاريخ.
* 📈 **تقارير مالية دقيقة:** تقرير أرباح وخسائر (P&L) متقدم يحسب الإيرادات والتكاليف وهوامش الربح تلقائياً.
* ⚡ **يعمل 100% أوفلاين:** بفضل تقنية Service Workers، يعمل التطبيق حتى عند انقطاع الإنترنت.
* 🔒 **خصوصية تامة:** البيانات تخزن في المتصفح (IndexedDB) باستخدام مكتبة Dexie.js، ولا يتم إرسالها لأي سيرفر خارجي.
* 💾 **نسخ احتياطي واستعادة:** إمكانية أخذ نسخة احتياطية من البيانات يدوياً لضمان عدم ضياعها.

---

## 🛠️ التقنيات المستخدمة

تم بناء هذا النظام باستخدام تقنيات الويب الأساسية والحديثة لضمان الخفة والسرعة:

* **واجهة المستخدم (Frontend):** HTML5, Vanilla JavaScript, Tailwind CSS (للتصميم المتجاوب).
* **قاعدة البيانات المحلية (Database):** [Dexie.js](https://dexie.org/) (مكتبة لتسهيل التعامل مع IndexedDB).
* **دعم العمل دون اتصال (Offline Support):** Service Workers & Web App Manifest (PWA).

قم بتنزيل ملفات المشروع على جهازك.
استخدم أي خادم محلي، مثل:
XAMPP / WAMP: ضع المجلد داخل htdocs وافتح الرابط http://localhost/MentraERPLiteOffline.
VS Code: استخدم إضافة Live Server.
بمجرد فتح التطبيق في المتصفح، سيتم تسجيل الـ Service Worker وحفظ الملفات في الكاش.
